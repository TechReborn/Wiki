Compressor

Compressor used to make Copper Plate from Copper Ingot. It is a Low Tier machine with input as 32 E\t and internal buffer for 1k Energy.

Recipe​

Crafting Table - Techreborn:compressor

Usage​

Compressor used as a crafting ingredient in several recipes:

  • Chemical Reactor
  • Implosion Compressor
  • Recycler
  • Rolling Machine

Main usage of Compressor is to make plates and compressed plant balls.

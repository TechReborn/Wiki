Saw Mill

The Saw Mill is a multi-block wood processing machine.

Recipe​

Crafting Table

Building instructions​

The Saw Mill multi-block has 3 layers:

  • First layer: 9x Basic Machine Casing
  • Second layer: 8x Advanced Machine Casing with a central hole filled with water
  • Third layer: 9x Basic Machine Casing

Place the Saw Mill block on the top and connect it to a power source.

Usage​

Put wood and water into the input slot. The Saw Mill will produce planks, and with a chance, some Saw Dust and paper.

# Industrial Centrifuge

The **Industrial Centrifuge** is a machine used to produce <McItem slug="techreborn:lava_cell" inline={true}/> and <McItem slug="techreborn:copper_dust" inline={true}/> from various items.

| Tier | Input   | Power Draw | Storage |
|------|---------|------------|---------|
| Low  | 32 E/t  | 5 E/t      | 10 kE   |

Used as an ingredient in the <McItem slug="minecraft:crafting_table"/> to produce:
<CraftingTable recipe=" input techreborn:industrial_centrifuge techreborn:energy_flow_circuit techreborn:industrial_centrifuge input techreborn:extractor techreborn:highly_advanced_machine_block techreborn:extractor input techreborn:industrial_electrolyzer techreborn:energy_flow_circuit techreborn:industrial_electrolyzer output techreborn:distillation_tower "/>

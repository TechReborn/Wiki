# Industrial Electrolyzer

**Industrial Electrolyzer** is used to produce <img src="/img/mods/techreborn/lava_cell.png" alt="lava_cell.png"/> **[Fluids](/items/fluid)** and <img src="/img/mods/techreborn/copper_dust.png" alt="copper_dust.png"/> **[Dusts](/items/dust)** from different items. It has an input rate of 512 FE/tick and internal buffer of 4000 FE.

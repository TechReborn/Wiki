Extractor

The Extractor is a machine that is used to process natural resources into another useful product. For example, Sap into Rubber.

The Extractor requires a power input of 32 E/tick to work and has an internal buffer of 1k E.

The extractor can also be used to discard the contents of a Cell so that you can use it again.

Crafting Table - Techreborn:extractor

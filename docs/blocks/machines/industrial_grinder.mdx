# Industrial Grinder

The **Industrial Grinder** is a multi-block machine that allows better grinding of items. It is a medium tier machine with 40000FE internal energy buffer and 256FE input. Also it has internal tank for 16 buckets which store fluid used to grind ore.

## Recipe

<CraftingTable recipe="input techreborn:industrial_electrolyzer techreborn:advanced_circuit techreborn:grinder input techreborn:diamond_grinding_head techreborn:diamond_grinding_head techreborn:diamond_grinding_head input techreborn:advanced_circuit techreborn:advanced_machine_block techreborn:advanced_circuit output techreborn:industrial_grinder"/>

## Building instructions

The **Industrial Grinder** multi-block has 3 layers.  
The first layer consists of 9x <McItem slug="techreborn:standard_machine_casing" inline={true}/>  
<img src="/img/blocks/industrial_grinder_layer1.png" alt="industrial_grinder_layer1.png"/>

The second layer consists of 8x <McItem slug="techreborn:advanced_machine_casing" inline={true}/> with a central hole filled with water.  
<img src="/img/blocks/industrial_grinder_layer2.png" alt="industrial_grinder_layer2.png"/>

The third layer again contains 9x <McItem slug="techreborn:standard_machine_casing" inline={true}/>  
<img src="/img/blocks/industrial_grinder_layer3.png" alt="industrial_grinder_layer3.png"/>

Attach the **Industrial Grinder** to the middle layer centre and connect it to a power source.  
<img src="/img/blocks/industrial_grinder.png" alt="industrial_grinder.png"/>

## Machine GUI

**Industrial Grinder** has the following elements on its GUI:

![GUI](/img/blocks/guiindustrialgrinder.png)

1. Four slots for upgrades  
2. Energy indicator. Click on it to switch energy display between EU and FE  
3. Input slot for cells (for draining or filling)  
4. Output slot for used cells  
5. Internal tank with tooltip info  
6. Input slot for ore  
7. Four output slots  
8. Hologram button  
9. JEI recipe button

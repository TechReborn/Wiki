# Compressor

**Compressor** used to make <McItem slug="techreborn:copper_plate" inline={true}/> from <McItem slug="techreborn:copper_ingot" inline={true}/>. It is a Low Tier machine with input as 32 E\t and internal buffer for 1k Energy.

## Recipe

<CraftingTable recipe=" input minecraft:stone air minecraft:stone input minecraft:stone techreborn:electronic_circuit minecraft:stone input minecraft:stone techreborn:basic_machine_frame minecraft:stone output techreborn:compressor "/>

## Usage

**Compressor** used as a crafting ingredient in several recipes:

- <McItem slug="techreborn:chemical_reactor"/>
- <McItem slug="techreborn:implosion_compressor"/>
- <McItem slug="techreborn:recycler"/>
- <McItem slug="techreborn:rolling_machine"/>

Main usage of **Compressor** is to make plates and compressed plant balls.

The **Distillation Tower** is a multiblock machine that is used to crack oil into components. The **Distillation Tower** is a medium tier machine with 128 E\t input rate and has an internal buffer of 10k Energy.

## Building Instructions

The **Distillation Tower** requires a 3×3×4 multiblock structure with a hollow center.

- **Layer 1**: 9x <McItem slug="techreborn:basic_machine_casing" inline={true}/>
- **Layer 2**: 8x <McItem slug="techreborn:industrial_machine_casing" inline={true}/>
- **Layer 3**: 8x <McItem slug="techreborn:basic_machine_casing" inline={true}/>
- **Top Layer**: 9x <McItem slug="techreborn:industrial_machine_casing" inline={true}/>

Attach the machine controller to the **bottom center block** and connect power to operate it.

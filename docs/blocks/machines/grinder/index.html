Grinder

The Grinder is a machine used to turn ores and other materials into dusts. It will not allow you to double ingot production from Raw Ores. Please, use Rock Cutter to get Ore Blocks and grind them in Grinder.

Mining Ore Blocks instead of Raw Ores will also allow you to triple ingot production in future using Industrial Grinder.

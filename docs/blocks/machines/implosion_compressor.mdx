# Implosion Compressor

The **Implosion Compressor** uses the raw power of TNT to make items. It is a multi-block machine with 64k energy buffer and 64 E\t input rate.

## Recipe

<CraftingTable recipe="input techreborn:advanced_alloy_ingot techreborn:advanced_machine_frame techreborn:advanced_alloy_ingot input techreborn:advanced_circuit techreborn:compressor techreborn:advanced_circuit input techreborn:advanced_alloy_ingot techreborn:advanced_machine_frame techreborn:advanced_alloy_ingot output techreborn:implosion_compressor"/>

## Building instructions

The **Implosion Compressor** multi-block has 3 layers. The **first layer** contains 9x <McItem slug="techreborn:advanced_machine_casing" inline={true}/> <img src="/img/blocks/layer1.png" alt="layer1.png"/><br/>
The **second layer** contains 8x <McItem slug="techreborn:advanced_machine_casing" inline={true}/> with a hole in the centre. <img src="/img/blocks/layer2.png" alt="layer2.png"/><br/>
The **third layer** is the same as the first layer. <img src="/img/blocks/layer3.png" alt="layer3.png"/><br/>
Put the **Implosion Compressor** on the top and provide some energy to it. <img src="/img/blocks/implosion_compressor.png" alt="implosion_compressor.png"/>

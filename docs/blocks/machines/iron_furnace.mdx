# Iron Furnace

An **Iron Furnace** works much like a traditional <McItem slug="minecraft:furnace" inline={true}/>, except that it is 56.25% more efficient with the fuel you use. You can see in the below recipes that both use the same amount of fuel, but the <McItem slug="techreborn:iron_furnace" inline={true}/> produced an extra 9 ingots compared to the traditional <McItem slug="minecraft:furnace" inline={true}/>.

\<recipe\> size 1x2 input minecraft:iron_ingot,16 input minecraft:coal,2 output techreborn:refined_iron_ingot,16 tool minecraft:furnace \</recipe\>\

\<recipe\> size 1x2 input minecraft:iron_ingot,25 input minecraft:coal,2 output techreborn:refined_iron_ingot,25 tool techreborn:iron_furnace \</recipe\>

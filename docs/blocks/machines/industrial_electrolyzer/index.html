Industrial Electrolyzer

Industrial Electrolyzer is used to produce Fluids and Dusts from different items. It has an input rate of 512 FE/tick and internal buffer of 4000 FE.

Recipe​

Crafting Table - Techreborn:industrial Electrolyzer

Industrial Blast Furnace

The Industrial Blast Furnace is a multi-block machine used to smelt tough metals. It has an input rate of 512 FE/tick and an internal buffer of 40,000 FE.
The smelting abilities depend on the casings used for the multi-block.

Building Instructions‚Äã

The Industrial Blast Furnace requires a 3√ó4 multi-block with a hole in the center.
The multi-block can be built from different machine casings. Possible options include:

  • Standard Machine Casing
  • Advanced Machine Casing
  • Industrial Machine Casing

This configuration will generate heat inside the furnace, enabling it to smelt the toughest ores (the type of casing used determines the heat generated).

  • First layer: 9x casings
  • Second layer: 8x casings with a hollow center
  • Third layer: 8x casings with a hollow center
  • Fourth layer: 9x casings

Place the Industrial Blast Furnace in the center of the bottom layer and connect it to a power source.

  • A multi-block built using Standard Machine Casing will generate 1360 heat
  • A multi-block built using Industrial Machine Casing will generate 3230 heat
tip

Two blocks of lava placed in the center of the multi-block will add 500 heat.

# Fusion Control Computer

The **Fusion Control Computer** is part of a multi-block generator that produces energy from various fluids. It is unique in that it does not simply consume the fuel, but instead processes them into a new item. It is also used as a machine to produce <McItem slug="techreborn:platinum_dust" inline={true}/> and <McItem slug="techreborn:iridium_ore" inline={true}/>.  
It must be combined with a minimum of 16 <McItem slug="techreborn:fusion_coil" inline={true}/>s to create a usable fusion reactor.

### Usage

- Adjustable hologram with H key
- Outputs power only while crafting
- Inputs energy only while idle
- Must be pre-charged before activation

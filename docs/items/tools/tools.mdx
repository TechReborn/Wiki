Tech Reborn provides wide variety of powered and not-powered tools for different needs.

## Un-powered tools

- Tree-tap
- Wrench
- Painting tool

## Powered tools

Powered tools serves as replacement for regular Minecraft tools and weapons like Pickaxe, Axe, Sword.

### Basic Tier

- Basic Chainsaw
- Basic Drill
- Basic Jackhammer
- Electric Tree tap

### Advanced Tier

- Advanced Chainsaw
- Advanced Drill
- Advanced Jackhammer
- Rock Cutter

### Industrial Tier

- Industrial Chainsaw
- Industrial Drill
- Industrial Jackhammer
- Nanosaber
- Omni-tool

There is also a creative-only Debug tools

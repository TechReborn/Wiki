**Advanced Chainsaw** is a more advanced chainsaw compared to <McItem slug="techreborn:basic_chainsaw" inline={true}/> available from Tech Reborn.

It has 100k Energy storage. One can charge it using energy storage blocks, like <McItem slug="techreborn:battery_box" inline={true}/> or any other machine capable of charging powered items. Charge rate will be 2048E\t.

Chainsaw could be used as a replacement for Diamond Axe, but break speed is much faster if powered. It will cost 100 Energy to break one block.

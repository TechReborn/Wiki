# Nickel Dust

**Nickel Dust** is a dust added by TechReborn, and is an important stepping stone to the more complicated machinery added by the mod. It is used to produce <McItem slug="techreborn:nickel_ingot" />.

Before the player makes the <McItem slug="techreborn:industrial_grinder" />, the primary way of acquiring <McItem slug="techreborn:nickel_dust" /> is from <McItem slug="techreborn:small_pile_of_nickel_dust" />.

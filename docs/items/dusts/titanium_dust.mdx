# Titanium Dust

**Titanium Dust** is a source for <McItem slug="techreborn:titanium_ingot" inline={true}/>, needed for many of the endgame machines.

The source of the first few <McItem slug="techreborn:titanium_dust" inline={true}/> will be crafting from <McItem slug="techreborn:small_pile_of_titanium_dust" inline={true}/>

Later, <McItem slug="techreborn:titanium_dust" inline={true}/> is best acquired by crafting from <McItem slug="techreborn:uu_matter" inline={true}/>

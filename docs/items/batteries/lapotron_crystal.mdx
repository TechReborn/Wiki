The **Lapotron Crystal** is a portable battery used to charge items. For more details and how to use, read [general information about Batteries](/docs/items/battery).

| Tier    | I/O Rate | Storage |
| ------- | -------- | ------- |
| Extreme | 2048 E/t | 10 ME   |

The **Lithium-Ion Battery** is a portable battery used to charge items. For more details and how to use, read [general information about Batteries](/docs/items/battery).

| Tier   | I/O Rate | Storage |
| ------ | -------- | ------- |
| Medium | 128 E/t  | 100 kE  |


## Recipe

```
<recipe> 
  size 1x2 
  input techreborn:aluminum_plate,2 
  input techreborn:lithium_cell,2 
  output techreborn:lithium_ion_battery 
  tool techreborn:assembling_machine 
</recipe>
```

## Usage

Used as an ingredient in the <McItem slug="minecraft:crafting_table"/> to produce:

- <McItem slug="techreborn:lithium_ion_batpack"/>
- <McItem slug="techreborn:advanced_chainsaw"/>

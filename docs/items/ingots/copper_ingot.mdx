# Copper Ingot

**Copper Ingot** is used to craft wires, alloys, etc. The main source for copper is <McItem slug="minecraft:Copper_Ore"/>.

## Recipe

#### Smelting Recipe

The main source of **Copper Ingot** is smelting <McItem slug="minecraft:Raw_Copper"/> or <McItem slug="minecraft:Copper_Ore"/> in any kind of Furnace, including the vanilla Furnace, <McItem slug="techreborn:iron_furnace"/> or <McItem slug="techreborn:electric_furnace"/>.

# Nickel Ingot

**Nickel Ingot** used to craft several recipes. Tech Reborn doesn't have Nickel ore. It could be obtained either by processing several ores in <McItem slug="techreborn:industrial_grinder"/> or by processing <McItem slug="minecraft:raw_copper"/>, <McItem slug="minecraft:raw_gold"/> or <McItem slug="minecraft:raw_iron"/> in <McItem slug="techreborn:industrial_centrifuge"/> if you do not have <McItem slug="techreborn:industrial_grinder"/> yet.

# Nickel Ingot

**Nickel Ingot** used to craft several recipes. Tech Reborn doesn't have Nickel ore. It could be obtained either by processing several ores in <McItem slug="techreborn:industrial_grinder" inline={true}/> or by processing <McItem slug="minecraft:raw_copper" inline={true}/>, <McItem slug="minecraft:raw_gold" inline={true}/> or <McItem slug="minecraft:raw_iron" inline={true}/> in <McItem slug="techreborn:industrial_centrifuge" inline={true}/> if you do not have <McItem slug="techreborn:industrial_grinder" inline={true}/> yet.

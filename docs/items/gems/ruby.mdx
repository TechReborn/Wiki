# Ruby

**Ruby** is a gem added by TechReborn. Can be obtained from <McItem slug="techreborn:ruby_ore" inline={true}/>. It is used to craft ruby armor set and also it could be grinded to <McItem slug="techreborn:ruby_dust" inline={true}/> for further processing.

<CraftingTable recipe="input air air air input air techreborn:block_of_ruby air input air air air output techreborn:ruby,9" />

**Ruby** could be made back from dust using raw power of TNT in \<mcitem\>techreborn:implosion_compressor\</mcitem\>:\
4x\<mcitem\>techreborn:ruby_dust\</mcitem\> + 16x\<mcitem\>minecraft:TNT\</mcitem\> -\> 3x\<mcitem\>techreborn:ruby\</mcitem\> + 12x\<mcitem\>techreborn:dark_ashes\</mcitem\>\

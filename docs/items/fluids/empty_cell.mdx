The **Empty Cell** is a fluid container added by Tech Reborn. It is used in the same way as <McItem slug="bucket" pack="minecraft" inline={true}/> with only difference is that cells with fluid can stack.

## Usage

Use it as a bucket. Also one or more **Empty Cell** required in recipes which produce filled cells.

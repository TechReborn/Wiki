The **Beryllium Cell** is a fluid cell added by TechReborn. You can get by using empty cell on Beryllium fluid block or from several machine recipes.

## Recipe

**Beryllium Cell** could be crafted either in Industrial Electorlyzer or in Fluid Replicator.
